# Beamer Theme for Soaringlab

_(Status: alpha)_

An unofficial beamer theme for Soaringlab.

```latex
%%% use theme https://hartwork.org/beamer-theme-matrix/
% \usetheme{Warsaw} % 蓝黑
% \usetheme{Ilmenau} % 蓝色
% \usetheme{Madrid} % 蓝紫色
\usetheme{CambridgeUS} % 红色

% 自定义设定放在最后
\usetheme{soaring}
```
